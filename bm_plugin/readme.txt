This plugin adds widget and custom post type. The widget will display custom post type with thumbnail images. \
\
== Description ==\
\
This useful plugin is designed to help users to add widget and custom post type to their Wordpress. After installing this plugin, the widget will display custom post type with thumbnail images. The purpose of this plugin is to complete a project for CCT460H5: Advanced Web Design, taught by professor Laurie Rauch at University of Toronto Mississauga. You may not use BM Plugin for any commercial use. \

\
